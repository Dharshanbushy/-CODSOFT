Introduction to the Titanic Dataset

The Titanic dataset encapsulates details of passengers aboard the ill-fated RMS Titanic. This historical dataset reveals insights into demographics, ticket class, and survival outcomes. Our goal is to leverage machine learning to predict survival probabilities based on these features, shedding light on the factors that influenced passengers' fate during this iconic maritime tragedy.

Description about the columns(variables) in the dataset.¶

PassengerId: Unique identifier for each passenger.

Survived: Binary variable indicating whether a passenger survived or not (0 = Not Survived, 1 = Survived).

Pclass: Ticket class, representing the socio-economic status of the passenger (1 = 1st class, 2 = 2nd class, 3 = 3rd class).

Name: Name of the passenger.

Sex: Gender of the passenger (male or female).

Age: Age of the passenger.

SibSp: Number of siblings/spouses aboard the Titanic.

Parch: Number of parents/children aboard the Titanic.

Ticket: Ticket number.

Fare: Fare paid for the ticket.

Cabin: Cabin number where the passenger stayed.

Embarked: Port of embarkation (C = Cherbourg, Q = Queenstown, S = Southampton).
